## 🚀 Projects That Actually Solve Problems

### 🏪 [Rental Management System](https://github.com/sriteja123gujjari/rental-management-system)
**React 19 · TypeScript · Supabase · Capacitor**

My family tracked rent in notebooks for years. I replaced that with a full-stack web + Android app.
- Multi-shop rent tracking with auto-split settlements
- Arrears detection, PDF reports, WhatsApp sharing
- Google OAuth + 3-step onboarding wizard
- **56 commits · Live on Vercel · Android APK released**

### 🗺️ [NagarSetu](https://github.com/sriteja123gujjari/nagarsetu)
**React Native · Firebase · Gemini Vision API · React Native Maps**

Civic issue reporting app — snap a photo, AI identifies the problem, pins it on the map.
- Gemini Vision API for photo-based issue classification
- Firebase real-time backend
- Google Maps integration for geo-tagged reports
